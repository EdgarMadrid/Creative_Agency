## ✨ Sobre el proyecto

**Creative_Agency** es un proyecto de práctica frontend orientado al desarrollo de una interfaz web moderna y visualmente atractiva.

Este proyecto fue desarrollado aplicando conocimientos adquiridos durante la formación de **Conquer Blocks**, enfocándose en mantener una estructura limpia y estilos reutilizables mediante **Variables CSS (Custom Properties)**.

Durante el desarrollo se trabajó en:

✅ Estructura semántica con HTML  
✅ Diseño y estilos con CSS moderno  
✅ Reutilización mediante Variables CSS  
✅ Organización visual por componentes  
✅ Buenas prácticas de desarrollo frontend  

---

## 🎯 Objetivo

Crear una interfaz moderna para una **Creative Agency** enfocándose en:

- Diseño limpio y atractivo
- Estilos reutilizables
- Mejor mantenimiento mediante Variables CSS
- Diseño responsive y organizado
