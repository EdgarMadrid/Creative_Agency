## ✨ About the Project

**Creative_Agency** is a frontend practice project focused on building a modern and visually appealing website interface.

This project was developed by applying concepts learned during **Conquer Blocks** training while emphasizing clean structure and maintainable styling using **CSS Custom Properties (Variables)**.

During development, the following concepts were applied:

✅ Semantic HTML structure  
✅ Styling with modern CSS  
✅ Reusable design through CSS Variables  
✅ Component-oriented layout  
✅ Frontend development best practices  

---

## 🎯 Goal

Create a modern **Creative Agency** interface focused on:

- Clean and attractive design
- Reusable styling
- Better maintainability through CSS Variables
- Responsive and organized layouts
